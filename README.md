I've gotten my hands dirty across JavaScript, TypeScript, ReactJs, NodeJS, C#, PHP, Python, Swift, Kotlin, Dart, and more. Most recently I was sole architect across 23 concurrent products at Ogoul Technology — the headline was re-architecting KT Messenger from 850 to 1.1M+ active users in 6 months, sub-2-second loads, full Signal Protocol E2EE. I'm CTO and founder of [Deep Voice AI Limited](https://deepvoice.ai), my own venture building full-duplex conversational voice agents. I'm also the creator of [DVAI-Bridge](https://github.com/dvai-global/dvai-bridge) — a patent-pending open-source local-AI inference substrate that unifies nine on-device inference engines behind a single OpenAI-compatible API across seven language SDKs. UK Patent Application GB2611312.6 pending, sole inventor.

I work end-to-end — frontend frameworks, backend microservices, real-time systems with WebSocket and Kafka, SQL and NoSQL databases, AWS and Azure cloud infrastructure, AI/ML pipelines with RAG, agentic systems, and edge inference. ### Highlights

- **DVAI-Bridge** ([github.com/dvai-global/dvai-bridge](https://github.com/dvai-global/dvai-bridge)) — Universal local inference substrate. Nine engines (llama.cpp, MediaPipe, MLX, Transformers.js, WebLLM, LiteRT, ONNX Runtime, ML.NET, llamafile) behind one OpenAI-compatible HTTP contract. Seven SDKs (TypeScript, Swift, Kotlin, Dart, C#, React Native, Capacitor). Transparent multi-device offload over LAN or self-hostable rendezvous. Patent-pending.
- **KT Messenger** — Full re-architecture, 850 → 1.1M+ users in 6 months, sub-2s loads, Signal Protocol E2EE, benchmarked faster than WhatsApp Web. [ktmessenger.com](https://ktmessenger.com)
- **PoshGPT Suite** — Generative AI platform at Ogoul: PoshGPT (LLM integration), PoshDetect (AI content moderation), PoshTranslate (multilingual AI). Integrated across 23 production products.
- **SocialBlox** — AI-powered MERN social platform with personalized recommendations, 8.2M active users, built by a 3-person team. Used by the Mexican Government and Election Commission.
- **L.I.N.K.S** — Offline-first voice AI assistant from 2012, custom ASR/TTS pre-Transformer. 280K paid users, zero marketing spend. [mega-voice-command.com](https://mega-voice-command.com)
